"우리에게 필요한 AI 스킬은 무엇인가요?"

AI 도입 미팅 때마다 가장 많이 받는 질문입니다.

그 답을 한 곳에 모아두려고 이 페이지를 만들었습니다. AI 서비스에서 바로 쓸 수 있는 스킬 중 비즈니스 리더에게 진짜 필요한 것만 엄선했습니다.

개발 전용 스킬은 최소화하고, 비즈니스·마케팅·리서치·투자 중심으로 직접 골랐습니다. 한국 특화 K-스킬도 별도 카테고리로 분리했습니다.

01

선정 기준

  • I비즈니스 리더가 바로 쓸 수 있을 것
  • II실무 검증이 끝났을 것
  • III한국 맥락에 맞을 것
02

왜 만들었나요

조직에 AI를 도입할 때 저는 늘 "전용 스킬 라이브러리부터 만드세요"라고 조언드립니다. 표준 스킬이 먼저 깔려야 도입 속도가 달라지기 때문입니다. 이 페이지는 그 레퍼런스입니다.

03

왜 'AI Roasting'인가요

좋은 원두를 고르고 로스팅해야 마실 만한 커피가 되듯, AI 스킬도 가려내고 다듬는 손이 필요합니다. 그 역할을 하겠다는 뜻입니다.

스킬 사용법

① 스킬 선택
원하는 스킬의 GitHub 링크를 열어 SKILL.md와 함께 들어 있는 파일을 확인하세요.
② 도구의 스킬 폴더에 복사
사용하는 AI 도구(Claude Code · Codex · Antigravity 등)의 스킬 폴더에 파일을 넣으면 자동으로 인식됩니다. 도구별 경로는 각 공식 문서를 참고하세요.
③ 조합해서 쓰기
스킬은 여러 개를 함께 써도 됩니다. 하네스로 팀을 짜고 워크플로우 스킬로 진행 방식을 잡으면 시너지가 커집니다.
